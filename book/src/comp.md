
# Components

Components are atomic units of data which may be used to represent
our entities. A drill would have:

- `position(x, y, 0)`
- `health(hp)`
- `tiling(size, 0)`
- ...


## Physics

- `Position`
  - `x`
  - `y`
  - `z`

- `Velocity`
  - `x`
  - `y`
  - `z`

- `Rotation(r)`

- `Mass(w)`

## Combat

- `Health(hp)`

- `Damage(hp)`

## Tiling

- `Block`
  - `size`
  - `rotation`

## Rendering

- `Sprite`
- `Vertices`
- `Shaders`

## Metadata

- `Name`
- `Description`
